Programa que calcula los exponentes promedio de Lyapunov
de un sistema dinámico de dos dimensiones.



Instrucciones:
==============

1.) Compilar y correr:
----------------------

cd src

make

cd ../bin/

./lyapunov


2.) Limpiar:
------------

cd src

make clean
